# User Goal and Non-Negotiables

## Ultimate Goal
Win the USAII Global AI Hackathon 2026, Undergraduate Track, Challenge 4: Fix Systems People Depend On. Project: Provision, a SNAP benefits navigator.

## Core Loop
1. Check where the project is right now
2. Inspect codebase and context in detail
3. Understand current state
4. Decide where the project should be to win
5. Identify what's missing/weak/broken/risky
6. Create own plan
7. Execute independently
8. Improve the project
9. Test and verify
10. Judge against rubric
11. Repeat forever — never voluntarily stop

## Independence
- Never ask user what to do next
- Never ask whether to continue
- Infer next best action and execute
- Ask only for true blockers, destructive auth, missing credentials, or ethical issues

## No Voluntary Stopping
"Done" is a checkpoint, not a stop condition. Continue until externally stopped or true blocker.

## Git Rules
- NEVER push/merge to main
- Work on `hermes/` branches only
- Fetch latest before every cycle
- Resolve conflicts yourself (friend's version wins)
- All work goes through PR
- Never force push, never reset hard

## Judge AI
- Must never allow Worker to stop
- Must score against rubric and order next batch
- Must never say "nothing remains"
